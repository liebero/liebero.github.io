### 02-4. 튜플 자료형 (괄호로 나타낸다.)


1. 인덱싱 하기



```python
t1 = (1, 2, 'a', 'b', 4, 5, 6)
t1[2]
```

<pre>
'a'
</pre>
2. 슬라이싱하기



```python
t1 = (1, 2, 'a', 'b', 4, 5, 6)
t1[2:7]
```

<pre>
('a', 'b', 4, 5, 6)
</pre>
3. 튜플 더하기



```python
t1 = (4)
t2 = (5)
a = t1 + t2
a
```

<pre>
9
</pre>
4. 튜플 곱하기



```python
t1 = ("사람")
t2 = (2)
a = (t1*t2)
a
```

<pre>
'사람사람'
</pre>
5. 튜플 길이 구하기



```python
t1 = ("사람이 되는 것이 중요하다.")
len(t1)
```

<pre>
15
</pre>
