### □ VSCode에서 깃허브 사용

#### 1. 깃허브 설치
sudo apt install git

#### 2. 사용자 메일과 이름 등록
git config --global user.email "메일주소@gmil.com"<br>
git config --global user.name "이름"

#### 3. 저장소 가지(브런치)를 메인으로 등록
git push --set-upstream origin main
